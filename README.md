### 简单安装
1.  git clone*https://github.com/slycmiaoxi/x-blog.git <br/>
2.  导入sql文件到数据库.<br/>
3.  项目依赖redis，fastdfs，Elasticsearch，ActiveMQ（ps：不是必须，可以按照功能取舍)<br/>
4.  配置和修改相关配置文件或者全局变量（后面有时间将详细优化后列出来)
